## 事件
2009年6月23日，A站宕机。当时的AcFun会员“bluesola”(徐逸)花了三天时间制作了一个名为“Mikufans”的网站，并在A站的百度贴吧宣传，声称是“AcFun后花园”。

然而在2010年AcFun“内乱”时，站内大量视频的弹幕中出现了“大陆最好的弹幕站bilibili.us”以及“大陆喷子最多的弹幕站acfun.cn”等抬高B站贬低A站的弹幕，导致大量会员流向B站。
